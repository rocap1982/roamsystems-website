# Session Logs

Append-only session logs. Each file covers one working session (1-10 bullets).

## Format

`YYYY-MM-DD.md` — one file per session day.

## Structure

- **Summary** (1-5 bullets)
- **Closed / shipped** (0-5 bullets)
- **Decisions / notes** (0-5 bullets)
- **Next / blockers** (0-5 bullets)
